Image Example

A basic map which uses an image to build a map layer.

<-- change png to gif if you are testing that aspect of the code>

<source>

The applet requires two parameters, the first 'image' points to the relative location of the image to display, while the second 'bounds' gives the spatial extent of the image.

Note: You only need to include jars/JimiCore.jar if you need to read and display images of types other than gif and jpeg.

<APPLET ARCHIVE="jars/collections.jar,jars/JimiCore.jar,jars/geotools.jar" CODE="ImageExample.class" WIDTH=430 HEIGHT=270>
<PARAM NAME="image" VALUE="maps/worldview.gif">
<PARAM NAME="bounds" VALUE="-180,-90,360,180">
</APPLET>
